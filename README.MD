# git-network

> git log graph overview (a copy from [github-network
](http://lcs.com))

**node.js is recommended**

## install
```bash
$ git clone res git-network
$ cd git-network
$ npm install
```

## usage

```bash
$ node app.js
```
- **brower on port 9696 (http://localhost:9696)**
- input a git repository path
